EffectiveGoggles

EG is my sideproject at the moment. I am trying to develop a CSS library that will allow me to quickly and efficiently develop a simple, clean, and accessible user interface. I will eventually be including code that will allow myself or other users to put predefined classes on the body that will change the theme of the page.

There is no live version of this project as of yet.

Divvy - A Business Oriented Carpool Organizer

Divvy was the final project I produced along with a classmate for my bootcamp. It aims to be a carpool organizer marketed towards companies to be used a service for employees, providing them a single, simple solution for organizing carpools and ridesharing with their coworkers.
I was mainly responsible for concept, design, and several parts of UI development.

ESEN - A Sports Tracker Dedicated to Professional E-Sports

ESEN was the second project I was involved in with my bootcamp. I was involved in concept, design, and the majority of UI development. This project is as of yet unfinished, as we ran out of time to work on this project. We did not initially understand all of the complexity involved in this project, and as such overestimated what we could get done in the period of time we had.
